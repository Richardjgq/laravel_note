### 模板与子模板

引入后台首页,创建layout模板继承

```
@extends('layout.admin')
@section('content')
# 中间的内容是可变的内容,其他内容继承自layout/admin
@endsection
```

使用`@yield('content')`设置继承

最后登陆成功,跳转使用redirect\(\)跳转

```
return redirect('admin/index');
```

系统基本信息中使用了几个常量和系统变量

```
<ul>
    <li>
        <label>操作系统</label><span>{{ PHP_OS }}</span>
    </li>
    <li>
        <label>运行环境</label><span>{{ $_SERVER['SERVER_SOFTWARE'] }}</span>
    </li>
    <li>
        <label>PHP版本</label><span>{{ PHP_VERSION }}</span>
    </li>
    <li>
        <label>软件版本</label><span>v-0.1</span>
    </li>
    <li>
        <label>上传附件限制</label><span>{{ get_cfg_var('upload_max_filesize') ? get_cfg_var('upload_max_filesize') : '不允许上传附件' }}</span>
    </li>
    <li>这里需要在config/app配置文件中设置时区为'timezone' => 'PRC'
        <label>北京时间</label><span>{{ date('Y年m月d日 H时i分s秒') }}</span>
    </li>
    <li>
        <label>服务器域名/IP</label><span>{{ $_SERVER['SERVER_NAME'] }} [ {{ $_SERVER['SERVER_ADDR'] }} ]</span>
    </li>
    <li>
        <label>Host</label><span>{{ $_SERVER['SERVER_ADDR'] }}</span>
    </li>
</ul>
```
