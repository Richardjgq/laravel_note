# 前端脚手架

### 知识点整理

```
1.默认使用NPM来安装管理前端包
    CSS:Laravel Mix的API用于编译SASS或Less
    JavaScript:使用Vue库让我们更轻松地编写现代JavaScript
2.编写CSS
    编辑package.json文件增删扩展包,例如里面的bootstrap-sass,jquery等等
    编译CSS之前,使用NPM安装应用的前端依赖:npm install
    可以使用Laravel Mix将SASS文件编译为原生的CSS
    npm run dev 命令会处理 webpack.mix.js 文件中的声明
    编译好的 CSS 文件会被放置到 public/css 目录下
    webpack.mix.js 文件会编译 SASS 文件 resources/assets/sass/app.scss
    配置Laravel Mix使用一个完全不同的预处理器
3.编写JavaScript
    package.json中包含了JS的依赖,也可以自定义扩展
    和CSS一样,npm install安装了扩展包之后,npm run dev编译前端资源
    在app.js 文件中你可以注册 Vue 组件或其他组件
    
    编写Vue组件
    在 app.js 中注册
    Vue.component('example', require('./components/Example.vue'));
    Blade模板就可以使用了
    @extends('layouts.app')

    @section('content')
        <example></example>
    @endsection
```
