# Centstead

虽然这个也不错,但还是用了自己搭建的VagrantBox.

这是一个Fork了Homestead的项目,功能几乎和Homestead一样.

**Centstead 预置软件： \(可基于配置快速切换版本\)**

* CentOS 7
* PHP 5.4 \/ 5.5 \/ 5.6 \/ 7.0
* Mysql 5.5 \/ 5.6 \/ 5.7
* MariaDB 5.5 \/ 10 \/ 10.1
* PostgreSQL 9.2 \/ 9.3 \/ 9.4 \/ 9.5
* Composer
* Nginx
* Redis
* Sqllite3
* Supervisor
* Xdebug
* Nodejs \(npm webpack gulp bower\)
* memcached
* Beanstalkd
